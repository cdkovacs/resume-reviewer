## Example

We are building a cloud platform engineering team to migrate a portfolio of
legacy Java and Node.js applications from on-prem VMs to Kubernetes on AWS.
The work includes writing Terraform modules for infrastructure provisioning,
building GitHub Actions CI/CD pipelines, containerizing applications, and
coaching application teams through the migration. Candidates should be
comfortable working directly with client stakeholders.
